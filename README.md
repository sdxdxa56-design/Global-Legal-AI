# Global-Legal-AI

## المهمة
الملفات الناقصه أو المجلدات الناقصه  معا كتابه الاكواد فيها  قم بتطوير المشروع بالكامل ليكون منصة لمستشار قانوني ذكي. قم بربط الـ API الخاص بنموذجي في ملف app.py، وأنشئ واجهة مستخدم (UI) احترافية بملفات HTML/CSS. أضف ملفاً جديداً باسم ideas.md يحتوي على 50 فكرة إبداعية لتطوير التكنولوجيا القانونية، واجعل الأكواد مترابطة وخالية من الأخطاء البرمجية 

## التشغيل
\`\`\`bash
pip install -r requirements.txt
python app.py
\`\`\`
